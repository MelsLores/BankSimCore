** IMPORTANTE: LEA ESTO PRIMERO **

Este sistema tiene 2 modos de uso:

===============================================================================
MODO 1: INTERFAZ WEB (RECOMENDADO) ‚≠ê
===============================================================================

Para usar la interfaz web moderna con todos los features:

Windows PowerShell:
  .\run_web.ps1

Windows Command Prompt:
  run_web.bat

Luego abrir navegador en: http://localhost:8080

VENTAJAS:
- Interfaz visual moderna
- Dashboard interactivo
- Todos los tests visualizados
- Facil de usar
- No requiere conocimientos de consola


===============================================================================
MODO 2: CONSOLA (VERSION CLASICA)
===============================================================================

Para usar el sistema por consola tradicional:

Windows PowerShell:
  .\run_system.ps1

Windows Command Prompt:
  run_system.bat

VENTAJAS:
- Mas rapido de iniciar
- No requiere navegador
- Consume menos recursos
